# 每日报告 JSON 规范

每个助手每天结束时生成一份报告，存放在 `daily-reports/YYYY-MM-DD/<subject>_report.json`。

## 文件命名
- `706_report.json` — 706 中国古代文学与古代汉语
- `807_report.json` — 807 中国现当代文学与评论写作
- `english_report.json` — 考研英语一

## JSON Schema

```json
{
  "subject": "706",                    // 科目代号
  "subject_name": "中国古代文学与古代汉语", // 科目全称
  "date": "2026-06-22",               // 复习日期
  "generated_at": "2026-06-22T22:30:00", // 生成时间 ISO 8601
  "phase": "第一阶段：基础搭建",       // 当前阶段
  "study_time": "2.5h",              // 总复习时长
  "study_time_breakdown": {           // 分模块时长（可选）
    "古代文学": "1.5h",
    "古代汉语": "1h"
  },
  "sections": [                       // 复习内容段落（朗读单位）
    {
      "id": "s1",                     // 段落唯一ID
      "heading": "诗经·六义",         // 段落标题
      "type": "论述题框架",           // 内容类型
      "content": "赋比兴是...",       // 正文（简洁概述，必填）
      "points": [                     // 层级结构（可选，用于思维导图渲染）
        {
          "text": "一级要点",
          "children": [
            { "text": "二级要点" },
            { "text": "另一个二级要点", "children": [...] }
          ]
        }
      ],
      "tags": ["#必考", "#先秦"],     // 标签
      "source": "袁行霈 第1卷 第3章"  // 出处（可选）
    }
  ],
  "key_review": [                     // 清晨回顾要点（3-5条）
    "诗经六义的定义与举例",
    "风雅颂的分类依据"
  ],
  "quiz": {                           // 测验结果（可选，无测验则 null）
    "total": 5,
    "correct": 4,
    "accuracy": "80%",
    "wrong_items": ["赋与比的区别"]
  },
  "weak_points": [                    // 薄弱点（0-N条）
    "赋比兴的具体举例区分"
  ],
  "tomorrow_plan": "古代汉语：宾语前置的类型与条件",
  "note": "今天诗经六义框架已建立..."  // 备注（可选）
}
```

## sections 内容类型

| type | 说明 |
|------|------|
| `论述题框架` | 完整的论述题答题框架 |
| `名词解释卡片` | 名词解释精简版 |
| `知识点梳理` | 知识点的理解和展开 |
| `真题练习` | 真题作答与解析 |
| `词汇积累` | 英语生词/短语 |
| `语法分析` | 长难句/语法点 |
| `评论写作` | 文学评论练笔 |
| `文本精读` | 教材原文精读摘要 |
| `测验` | 测验题目与结果 |

## 朗读规则（前端实现）

- 每个 `section` 渲染为一个卡片，右上角有 🔊 播放按钮
- 点击播放按钮朗读 `heading + content`
- 使用 Web Speech API（iOS Safari 内置，无需联网）
- 朗读时按钮变为 ⏸️ 暂停状态
