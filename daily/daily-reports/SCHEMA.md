# 每日晨报 JSON 规范

每天每个科目生成一份晨报，存入 `daily-reports/YYYY-MM-DD/<subject>_report.json`。

晨报的核心定位：**次日清晨的音频复习材料**，不是学习日志。内容按知识点组织，适合朗读收听。

## 文件命名
- `706_report.json` — 706 中国古代文学与古代汉语
- `807_report.json` — 807 中国现当代文学与评论写作
- `english_report.json` — 考研英语一

## JSON Schema

```json
{
  "subject": "706",
  "subject_name": "中国古代文学与古代汉语",
  "date": "2026-06-22",
  "generated_at": "2026-06-22T23:30:00",
  "audio_duration": "4min",
  "topics": [
    {
      "id": "t1",
      "title": "杜甫：沉郁顿挫与诗史精神",
      "summary": "杜甫是706最高频考点，核心三标签：沉郁顿挫、诗史、律诗",
      "points": [
        {
          "text": "生平四期：读书壮游→困守长安→陷贼为官→漂泊西南",
          "children": [
            {"text": "第一期（35岁前）：裘马轻狂，结交李白"},
            {"text": "第二期（35-44岁）：困守长安，仕途失意"},
            {"text": "第三期（44-48岁）：陷贼为官，安史之乱"},
            {"text": "第四期（48-59岁）：漂泊西南，夔州律诗巅峰"}
          ]
        }
      ]
    }
  ],
  "key_takeaways": [
    "杜甫三标签：沉郁顿挫(风格)、诗史(内容)、律诗(形式)——论述题三选一展开",
    "秋兴八首是七律巅峰，八首有机整体不可割裂"
  ],
  "quiz": {
    "total": 5,
    "correct": 4,
    "accuracy": "80%",
    "wrong_items": ["赋与比的区别"]
  },
  "weak_points": [
    "赋比兴的具体举例区分"
  ],
  "tomorrow_note": "诗经六义与艺术成就（T0考点，6次真题）"
}
```

## 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `subject` | ✓ | 科目代号：706 / 807 / english |
| `subject_name` | ✓ | 科目全称 |
| `date` | ✓ | 复习日期 YYYY-MM-DD |
| `generated_at` | ✓ | 生成时间 ISO 8601 |
| `audio_duration` | | 预估朗读时长，如 "4min" |
| `topics` | ✓ | 知识点单元数组（朗读+展示的基本单位） |
| `key_takeaways` | ✓ | 必记要点 3-5 条（音频末尾朗读） |
| `quiz` | | 测验结果，无则省略 |
| `weak_points` | | 薄弱点，无则空数组 |
| `tomorrow_note` | | 次日计划，一句话 |

### topics 结构

每个 topic 是一个**完整知识主题**，不是活动记录：

| 字段 | 说明 |
|------|------|
| `id` | 唯一标识，如 "t1" |
| `title` | 主题标题，精炼概括（朗读时会念出） |
| `summary` | 一句话概述，仅用于首页展示，不朗读 |
| `points` | 层级知识树，见下方 |

### points 层级结构（朗读核心）

```json
{
  "text": "一级要点（朗读时作为主干句）",
  "children": [
    {
      "text": "二级要点（细节展开）",
      "children": [
        {"text": "三级要点（具体例证）"}
      ]
    }
  ]
}
```

- 朗读时一级间停顿 `。。。`（约600ms），子级间停顿 `。。`（约400ms）
- 标题与正文间停顿 `。。。。。`（约1000ms）
- 深度建议 2-3 层，不超过 4 层（太深朗读效果差）

## 内容写作原则

1. **从"做了什么"转向"学会了什么"**：topic title 应该是知识点名，不是活动描述
   - ❌ "杜甫 · 教材原文精读" / "杜甫 · 秋兴八首逐首解析"
   - ✓ "杜甫：沉郁顿挫与诗史精神" / "秋兴八首：结构与意象系统"
2. **每个 topic 是一个自包含的知识单元**：脱离上下文也能听懂
3. **points 是朗读文本**：写的时候就想象它是被念出来的，不是被阅读的
4. **key_takeaways 是这科今天的"金句"**：次日早上最需要记住的 3-5 条
5. **控制篇幅**：三科合计朗读 10-15 分钟，每科约 3-5 分钟

## 音频生成

- 每科每天生成一个 mp3：`audio/706.mp3`、`audio/807.mp3`、`audio/english.mp3`
- 中文语音：`zh-CN-YunxiNeural`（云希男声）
- 英语语音：`en-US-GuyNeural`（Guy 男声）
- manifest 文件：`audio/manifest.json`，记录各 mp3 的 hash 用于增量更新
